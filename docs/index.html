Etimología

https://stacspec.org/

The SpatioTemporal Asset Catalog (STAC) specification provides a common language to describe a range of geospatial information, so it can more easily be indexed and discovered. A 'spatiotemporal asset' is any file that represents information about the earth captured in a certain space and time.

https://stacspec.org/

Conceptos

El catalogo contiene Collections y Items

Collection: Grupo de items (imagenes) que comparten las mismas características (sensor, nivel de procesamiento, etc)

Item: Una escena, un tile, un granule, etc. Un item pertenece a una y una sola collection

Ejemplos de Collections

  • Landsat 5 TOA reflectance
  • Landsat 5 procesado con el sistema ESPA
  • Landsat 8 procesado con el sistema ESPA
  • Sentinel 2 nivel L2A

Componentes

Ministac se divide en tres componentes

  • ministac: define la estructura de la base de datos y permite su acceso
  • ministac-api: JSON REST API para consultar el catalogo con consultas HTTP
  • ministac-admin: modulo de Administración, para agregar y actualizar las collections y items

Manejo del catalogo

Requiere

  • Un administrador sistema: Levanta el catalogo
  • Un data manager: actualiza el catalogo después de cada descarga de nuevos datos

cd ministac-api
docker-compose up -d db
docker-compose run --rm --entrypoint "python3" flaskapp -c "from ministac.db import init_db; init_db()"
docker-compose up -d

Usando la biblioteca de ministac-admin


add_collection.py collections/landsat_sr_8.yaml
add_items.py /path/to/scenes --pattern \"LC08.*\" --collection landsat_sr_8 --parser landsat
